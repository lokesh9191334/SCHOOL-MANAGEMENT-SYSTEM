Help & Support

Getting Started

Welcome to School Management System! Follow these steps to get started quickly.

1
Login to System

Use your email and password to login to the system.

2
Explore Dashboard

View your dashboard to see statistics and quick actions.

3
Add Students

Navigate to Students section and add new students.

4
Manage Classes

Create classes and assign students to them.

Student Management

1
Add New Student

Go to Students section → Click "Add Student" → Fill in required information (name, email, phone, class, etc.) → Click Save

2
Edit Student Information

Navigate to Students list → Click on student name → Click "Edit" → Update information → Save changes

3
View Student Details

Students list → Click on student name → View complete profile including attendance, fees, and performance

Attendance Management

1
Mark Daily Attendance

Attendance section → Select date → Select class → Mark present/absent for each student → Submit

2
View Attendance Reports

Attendance section → Select date range → Select class → Click "Generate Report" → View or export

3
Update Attendance

Attendance reports → Find the date → Click "Edit" → Update attendance status → Save changes

Fee Management

1
Create Fee Structure

Fees section → Click "Add Fee" → Select student → Enter amount and due date → Save

2
Record Payment

Fees list → Find student fee → Click "Pay Now" → Enter payment details → Save payment

3
Generate Fee Reports

Fees section → Click "Reports" → Select date range → Generate report → Export PDF/Excel

Common Issues & Solutions

Forgot Password?

Contact administrator to reset your password. You can also use the "Forgot Password" link on login page.

Data Not Updating?

Refresh the page or clear browser cache. If issue persists, contact technical support.

Reports Not Downloading?

Check browser download settings and ensure pop-ups are allowed for this site.

Mobile Access Issues?

Use mobile browser or PWA app. Ensure you have stable internet connection.
